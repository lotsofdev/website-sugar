# adoptStyleInShadowRoot
<div class="_metas">
<div class="_type"><span class="_type-label">Type:</span><span class="_type-value">Function</span></div>
<div class="_status"><span class="_status-label">Status:</span><span class="_status-value -stable">stable</span></div>
<div class="_since"><span class="_since-label">Since:</span><span class="_since-value">2.0.0</span></div>

<div class="_namespace">@lotsof.sugar.js.css</div>
<div class="_description">
This function allows you to make a shadowRoot element adopt his host context styles
</div>
<div class="_params">
## Params
<ol class="_list">
<li class="_item">
<span class="_name">$shadowRoot<span class="_required">*</span></span><span class="_default"></span> <span class="_type">HTMLShadowRootElement</span>
<p class="_description">The shadow root you want to adopt the &dollar;context styles</p>
</li>
<li class="_item">
<span class="_name">$context</span><span class="_default">document</span> <span class="_type">HTMLElement</span>
<p class="_description">The context from which you want to adopt the styles</p>
</li>
</ol>
</div>
<div class="_return">
## Return
<ol class="_list">
<li class="_item">
<span class="_description">Return a promise fullfilled when the styles have been adopted</span><span class="_default"></span><span class="_type">Promise</span>
</li>
</ol>
</div>
<div class="_examples">
## Example
```js
import { __adoptStyleInShadowRoot } from '@coffeekraken/sugar/dom';
const myShadowRoot = $myElement.shadowRoot;
await __adoptStyleInShadowRoot(myShadowRoot);
```
