# reloadStylesheets
<div class="_metas">
<div class="_type"><span class="_type-label">Type:</span><span class="_type-value">Function</span></div>
<div class="_status"><span class="_status-label">Status:</span><span class="_status-value -stable">stable</span></div>

<div class="_platform"><span class="_platform-label">Platform:</span><span class="_platform-value -js">js</span></div>

<div class="_namespace">@lotsof.sugar.js.dom.css</div>
<div class="_description">
This function just loop on each &quot;link&quot; tags that point to some css files
and reload them by adding a &quot;queryString&quot; with the timestamp in it
to prevent caching
</div>
<div class="_params">
## Params
<ol class="_list">
<li class="_item">
<span class="_name">$root</span><span class="_default">document</span> <span class="_type">HTMLElement</span>
<p class="_description">The root element in which to reload the stylesheets</p>
</li>
</ol>
</div>
<div class="_examples">
## Example
```js
import { __reloeadStylesheets } from '@lotsof/sugar/dom';
__reloadStylesheets();
```
