# whenImagesLoaded
<div class="_metas">
<div class="_type"><span class="_type-label">Type:</span><span class="_type-value">Function</span></div>
<div class="_status"><span class="_status-label">Status:</span><span class="_status-value -stable">stable</span></div>
<div class="_since"><span class="_since-label">Since:</span><span class="_since-value">2.0.0</span></div>
<div class="_platform"><span class="_platform-label">Platform:</span><span class="_platform-value -js">js</span></div>
</div>
<div class="_namespace">@lotsof.sugar.js.dom.detect</div>
<div class="_description">
Detect when some images are loaded. This function take advantage of the SPromise class
and trigger an event called &quot;img.loaded&quot; that will be triggered on each loaded images and another called &quot;loaded&quot; once all the images are loaded.
See in the example bellow.
</div>
<div class="_params">
## Params
<ol class="_list">
<li class="_item">
<span class="_name">$imgs<span class="_required">*</span></span><span class="_default"></span> <span class="_type">Array&lt;HTMLImageElement&gt;</span>
<p class="_description">An array &lpar;or nodeList&rpar; of HTMLImageElement to detect the load</p>
</li>
</ol>
</div>
<div class="_return">
## Return
<ol class="_list">
<li class="_item">
<span class="_description">A promise resolved when all images are loaded properly</span><span class="_default"></span><span class="_type">Promise</span>
</li>
</ol>
</div>
<div class="_examples">
## Example
```js
import { __whenImagesLoaded } from '@lotsof/sugar/dom'
__whenImagesLoaded([
	$img1, $img2, $img3
]).on('loaded', $img => {
   // do something with the loaded image
}).then(imgs => {
  // do something here
})
```
</div>
