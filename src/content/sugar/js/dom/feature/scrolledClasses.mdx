# scrolledClasses
<div class="_metas">
<div class="_type"><span class="_type-label">Type:</span><span class="_type-value">Function</span></div>
<div class="_status"><span class="_status-label">Status:</span><span class="_status-value -stable">stable</span></div>
<div class="_since"><span class="_since-label">Since:</span><span class="_since-value">2.0.0</span></div>
<div class="_platform"><span class="_platform-label">Platform:</span><span class="_platform-value -js">js</span></div>
</div>
<div class="_namespace">@lotsof.sugar.js.dom.feature</div>
<div class="_description">
This feature allows you to have `scrolled`, `scrolled-x` and `scrolled-y` classes added and removed from
the body depending on an scroll offset that can be set by:
1. Passing some offsets through the settings
2. Setting the `--s-scrolled-classes-{setting}` css variable in your style
</div>
<div class="_params">
## Params
<ol class="_list">
<li class="_item">
<span class="_name">settings</span><span class="_default">&lsqb;object Object&rsqb;</span> <span class="_type">TScrolledClassesSettings</span>
<p class="_description">The settings you want to override</p>
</li>
</ol>
</div>
<div class="_examples">
## Example
```js
import { __scrolledClasses } from '@lotsof/sugar/features';
__scrolledClasses({
     class: 'hello',
     offset: 200
});
```
</div>
<div class="_settings">
## Settings
<ol class="_list">
<li class="_item">
<span class="_name">offset</span><span class="_default">100</span> <span class="_type">number</span>

<li class="_item">
<span class="_name">offsetX</span><span class="_default"></span> <span class="_type">number</span>

</li>
<li class="_item">
<span class="_name">offsetY</span><span class="_default"></span> <span class="_type">number</span>

<li class="_item">

</li>
</ol>
