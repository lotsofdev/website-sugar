# nearestElement
<div class="_metas">
<div class="_type"><span class="_type-label">Type:</span><span class="_type-value">Function</span></div>
<div class="_status"><span class="_status-label">Status:</span><span class="_status-value -stable">stable</span></div>
<div class="_since"><span class="_since-label">Since:</span><span class="_since-value">2.0.0</span></div>
<div class="_platform"><span class="_platform-label">Platform:</span><span class="_platform-value -js">js</span></div>
</div>
<div class="_namespace">@lotsof.sugar.js.dom.query</div>
<div class="_description">
Get the nearest element from the passed one from the passed elements list.
You can pass a direction like &quot;top&quot;, &quot;right&quot;, &quot;bottom&quot;, &quot;left&quot; to specify
the direction you want to search the nearest element.
</div>
<div class="_params">
## Params
<ol class="_list">
<li class="_item">
<span class="_name">$from<span class="_required">*</span></span><span class="_default"></span> <span class="_type">HTMLElement</span>
<p class="_description">The element from which to start the search</p>
</li>
<li class="_item">
<span class="_name">$elements<span class="_required">*</span></span><span class="_default"></span> <span class="_type">HTMLElement&lsqb;&rsqb;</span>
<p class="_description">The elements list to search in</p>
</li>
<li class="_item">
<span class="_name">settings</span><span class="_default">&lsqb;object Object&rsqb;</span> <span class="_type">TNearestElementSettings</span>
<p class="_description">Some settings to configure your search</p>
</li>
</ol>
</div>
<div class="_return">
## Return
<ol class="_list">
<li class="_item">
<span class="_description">The nearest element found</span><span class="_default"></span><span class="_type">HTMLElement</span>
</li>
</ol>
</div>
<div class="_examples">
## Example
```js
import { __nearestElement } from '@lotsof/sugar/dom'
__nearestElement(myCoolElement, document.querySelectorAll('.my-cool-elements'), {
     direction: 'top'
});
```
