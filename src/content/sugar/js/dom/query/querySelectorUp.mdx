# querySelectorUp
<div class="_metas">
<div class="_type"><span class="_type-label">Type:</span><span class="_type-value">Function</span></div>
<div class="_status"><span class="_status-label">Status:</span><span class="_status-value -stable">stable</span></div>
<div class="_since"><span class="_since-label">Since:</span><span class="_since-value">2.0.0</span></div>
<div class="_platform"><span class="_platform-label">Platform:</span><span class="_platform-value -js">js</span></div>
</div>
<div class="_namespace">@lotsof.sugar.js.dom.query</div>
<div class="_description">
Go up the dom three to find the first element that matches the passed selector
</div>
<div class="_params">
## Params
<ol class="_list">
<li class="_item">
<span class="_name">$elm<span class="_required">*</span></span><span class="_default"></span> <span class="_type">HTMLElement</span>
<p class="_description">The element to start on</p>
</li>
<li class="_item">
<span class="_name">selector<span class="_required">*</span></span><span class="_default"></span> <span class="_type">String&vert;Function</span>
<p class="_description">A css selector to search for or a check function that will be used</p>
</li>
</ol>
</div>
<div class="_return">
## Return
<ol class="_list">
<li class="_item">
<span class="_description">The element found or null</span><span class="_default"></span><span class="_type">HTMLElement</span>
</li>
</ol>
</div>
<div class="_examples">
## Example
```js
import { __querySelectorUp } from '@lotsof/sugar/dom'
const closestElm =  __querySelectorUp(myCoolElement, '.my-cool-class');
if (closestElm) {
		// we have found en element that matches the selector
}
// the selector param can be a function that need to return either true or false like so:
 __querySelectorUp(myCoolElement, (elm) => {
  return elm.hasAttribute('my-cool-attribute')
})
```
