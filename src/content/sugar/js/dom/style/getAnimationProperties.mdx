# getAnimationProperties
<div class="_metas">
<div class="_type"><span class="_type-label">Type:</span><span class="_type-value">Function</span></div>
<div class="_status"><span class="_status-label">Status:</span><span class="_status-value -beta">beta</span></div>

<div class="_description">
Get the css animation properties from an HTMLElement in an object format
</div>
<div class="_params">
## Params
<ol class="_list">
<li class="_item">
<span class="_name">elm<span class="_required">*</span></span><span class="_default"></span> <span class="_type">HTMLElement</span>
<p class="_description">The element to get the properties from</p>
</li>
</ol>
</div>
<div class="_return">
## Return
<ol class="_list">
<li class="_item">
<span class="_description">The animation properties</span><span class="_default"></span><span class="_type">Object</span>
</li>
</ol>
</div>
<div class="_examples">
## Example
```js
import { __getAnimationProperties } from '@lotsof/sugar/dom'
const props = __getAnimationProperties(myCoolHTMLElement);
// output format
// {
// 	name : ['animation1'],
// 	duration : [200],
// 	delay : [0],
// 	timingFunction : ['linear'],
// 	iterationCount : [1],
// 	direction : ['forward'],
// 	totalDuration : 200
// }
```
