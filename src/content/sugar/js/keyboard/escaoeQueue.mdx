# escaoeQueue
<div class="_metas">
<div class="_type"><span class="_type-label">Type:</span><span class="_type-value">Function</span></div>
<div class="_status"><span class="_status-label">Status:</span><span class="_status-value -stable">stable</span></div>
<div class="_since"><span class="_since-label">Since:</span><span class="_since-value">2.0.0</span></div>
<div class="_platform"><span class="_platform-label">Platform:</span><span class="_platform-value">js</span></div>
</div>
<div class="_namespace">@lotsof.sugar.js.keyboard</div>
<div class="_description">
This funciton allows you to register actions to execute when user press the escape key.
It will take care of executing the last registered action first, then the others...
This function returns a SPromise instance on which you can call the `cancel` method to unregister your
action in the queue.
</div>

</li>
<li class="_item">

<p class="_description">An option object to configure your hotkey&period; Here&apos;s the list of available settings&colon;</p>

</div>

<ol class="_list">

</li>
</ol>
</div>
<div class="_examples">
## Example
```js
import { __escapeQueue } from '@coffeekraken/sugar/keyboard'
const promise = __escapeQueue();
promise.then(() => {
     // do something...
});

// if you want to cancel your subscription
promise.cancel();
```
