# replaceTokens
<div class="_metas">
<div class="_type"><span class="_type-label">Type:</span><span class="_type-value">Function</span></div>
<div class="_status"><span class="_status-label">Status:</span><span class="_status-value -stable">stable</span></div>
<div class="_since"><span class="_since-label">Since:</span><span class="_since-value">2.0.0</span></div>
<div class="_platform"><span class="_platform-label">Platform:</span><span class="_platform-value -js">js</span><span class="_platform-value -node">node</span></div>
</div>
<div class="_namespace">@lotsof.sugar.js.string</div>
<div class="_description">
This function takes as parameter a tokened string like &quot;something [cool]&quot;, an object
of arguments/values and return the processed string with the tokens replaced by the arguments values.
</div>
<div class="_params">
## Params
<ol class="_list">
<li class="_item">
<span class="_name">string<span class="_required">*</span></span><span class="_default"></span> <span class="_type">String</span>
<p class="_description">The string to process</p>
</li>
<li class="_item">
<span class="_name">tokensObj<span class="_required">*</span></span><span class="_default"></span> <span class="_type">Object</span>
<p class="_description">The arguments&sol;value object</p>
</li>
<li class="_item">
<span class="_name">settings</span><span class="_default">&lsqb;object Object&rsqb;</span> <span class="_type">Object</span>
<p class="_description">A settings object to configure the parsing process</p>
</li>
</ol>
</div>
<div class="_return">
## Return
<ol class="_list">
<li class="_item">
<span class="_description">The processed string</span><span class="_default"></span><span class="_type">String</span>
</li>
</ol>
</div>
<div class="_examples">
## Example
```js
import { __replaceTokens } from '@lotsof/sugar/string';
__replaceTokens('hello [world]', { world: 'Coco' }); // => hello Coco
```
</div>

</ol>
