# checkPathWithMultipleExtensions
<div class="_metas">
<div class="_type"><span class="_type-label">Type:</span><span class="_type-value">Function</span></div>
<div class="_status"><span class="_status-label">Status:</span><span class="_status-value -stable">stable</span></div>
<div class="_since"><span class="_since-label">Since:</span><span class="_since-value">2.0.0</span></div>
<div class="_platform"><span class="_platform-label">Platform:</span><span class="_platform-value -node">node</span></div>
</div>
<div class="_namespace">@lotsof.sugar.node.fs</div>
<div class="_description">
This function take a path and some extensions to check if a file
exists with one of these particular extensions.
If a file exists, the function return the path with the first extensions that matches
</div>
<div class="_params">
## Params
<ol class="_list">
<li class="_item">
<span class="_name">path<span class="_required">*</span></span><span class="_default"></span> <span class="_type">String</span>
<p class="_description">The file path you want to check&period; With or without an extension</p>
</li>
<li class="_item">
<span class="_name">extensions<span class="_required">*</span></span><span class="_default"></span> <span class="_type">Array&lt;String&gt;</span>
<p class="_description">The extensions &lpar;without the dot&rpar; you want to check</p>
</li>
</ol>
</div>
<div class="_return">
## Return
<ol class="_list">
<li class="_item">
<span class="_description">The first valid path founded, or undefined</span><span class="_default"></span><span class="_type">String&vert;undefined</span>
</li>
</ol>
</div>
<div class="_examples">
## Example
```js
import { __checkPathWithMultipleExtensions } from '@lotsof/sugar/fs';
__checkPathWithMultipleExtensions('/my/cool/file.txt', ['txt','js','css']);
```
