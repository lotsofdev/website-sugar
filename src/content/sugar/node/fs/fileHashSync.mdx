# fileHashSync
<div class="_metas">
<div class="_type"><span class="_type-label">Type:</span><span class="_type-value">Function</span></div>
<div class="_status"><span class="_status-label">Status:</span><span class="_status-value -stable">stable</span></div>
<div class="_since"><span class="_since-label">Since:</span><span class="_since-value">2.0.0</span></div>
<div class="_platform"><span class="_platform-label">Platform:</span><span class="_platform-value -node">node</span></div>
</div>
<div class="_namespace">@lotsof.sugar.node.fs</div>
<div class="_description">
This function allows you to get back an integrity hash for the passed file.
This mean that if a file returns the same integrity hash twice, the folder or files in it
has not been updated...
</div>
<div class="_params">
## Params
<ol class="_list">
<li class="_item">
<span class="_name">filePath<span class="_required">*</span></span><span class="_default"></span> <span class="_type">String</span>
<p class="_description">The folder path you want to get the hash back</p>
</li>
<li class="_item">
<span class="_name">settings</span><span class="_default">&lsqb;object Object&rsqb;</span> <span class="_type">IFileHashSettings</span>
<p class="_description">Some settings to configure your hash generation process</p>
</li>
</ol>
</div>
<div class="_return">
## Return
<ol class="_list">
<li class="_item">
<span class="_description">The calculated folder hash</span><span class="_default"></span><span class="_type">String</span>
</li>
</ol>
</div>
<div class="_examples">
## Example
```js
import { __fileHashSync } from '@lotsof/sugar/fs';
__fileHashSync('my/cool/folder'); // => YZOrKDx9LCLd8X39PoFTflXGpRU=,
```
</div>
