  # folderHashSync

      <span class="_type-label">Type:</span>

    <div class="_status">

  <div class="_namespace">@lotsof.sugar.node.fs</div>
  <div class="_description">
    This function allows you to get back an integrity hash for the passed
    folder. This mean that if a folder returns the same integrity hash twice,
    the folder or files in it has not been updated...
  </div>
  <div class="_params">
    ## Params
    <ol class="_list">
      <li class="_item">
        <span class="_name">
          folderPath<span class="_required">*</span>
        </span>
        <span class="_default"></span> <span class="_type">String</span>
        <p class="_description">
          The folder path you want to get the hash back
        </p>
      </li>
      <li class="_item">
        <span class="_name">settings</span>
        <span class="_default">&lsqb;object Object&rsqb;</span> <span class="_type">
          TFolderHashSettings
        </span>
        <p class="_description">
          Some settings to configure your hash generation process
        </p>
      </li>
    </ol>
  </div>
  <div class="_return">
    ## Return
    <ol class="_list">
      <li class="_item">
        <span class="_description">The calculated folder hash</span>
        <span class="_default"></span>
        <span class="_type">String</span>
      </li>
    </ol>
  </div>
  <div class="_examples">
    ## Example ```js import {__folderHashSync} from '@lotsof/sugar/fs';
    __folderHashSync('my/cool/folder'); // => YZOrKDx9LCLd8X39PoFTflXGpRU=, ```
  </div>
