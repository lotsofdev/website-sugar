# deepFilter
<div class="_metas">
<div class="_type"><span class="_type-label">Type:</span><span class="_type-value">Function</span></div>
<div class="_status"><span class="_status-label">Status:</span><span class="_status-value -beta">beta</span></div>

<div class="_namespace">@lotsof.sugar.php.object</div>
<div class="_description">
This function allows you to filter deeply an object or an array.
It will call the $callback closure on higher elements first then will go deeper on left items
</div>
<div class="_params">
## Params
<ol class="_list">
<li class="_item">
<span class="_name">$value<span class="_required">*</span></span><span class="_default"></span> <span class="_type">Array</span>
<p class="_description">The array&sol;object to deep filter on</p>
</li>
<li class="_item">
<span class="_name">$callback<span class="_required">*</span></span><span class="_default"></span> <span class="_type">Function</span>
<p class="_description">The function to run on each values&period; If return true&comma; the item is kept&comma; if false&comma; it is removed</p>
</li>
</ol>
</div>
<div class="_return">
## Return
<ol class="_list">
<li class="_item">
<span class="_description">The processed array/object</span><span class="_default"></span><span class="_type">Array</span>
</li>
</ol>
</div>
<div class="_examples">
## Example
```php
\Sugar\object\deepFilter((object) [
   'prop1' => 'Hello',
   'prop2' => 'World'
], function($prop, $value) {
     return false;
});

```
