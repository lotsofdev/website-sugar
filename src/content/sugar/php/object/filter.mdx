# filter
<div class="_metas">
<div class="_type"><span class="_type-label">Type:</span><span class="_type-value">Function</span></div>
<div class="_status"><span class="_status-label">Status:</span><span class="_status-value -beta">beta</span></div>

<div class="_namespace">@lotsof.sugar.php.object</div>
<div class="_description">
This function allows you filter some object items either by:
- `^...`: The keys that starts with the passed string
- `...$`: The keys that ends with the passed string
- `function($key, $value) { retur true; }`: A function that must return true or false
</div>

<div class="_examples">
## Example
```php
$obj = (object) [
  "hello" => 'world',
  "yop" => 'plop',
  "helloPlop" => true
];
\Sugar\object\filter($obj, '^hel'); // (object) ["hello" => 'world', "helloPlop" => true]
\Sugar\object\filter($obj, function($key, $value) {
  return false;
}); // (object) []
```
