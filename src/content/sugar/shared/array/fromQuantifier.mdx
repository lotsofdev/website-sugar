# fromQuantifier
<div class="_metas">
<div class="_type"><span class="_type-label">Type:</span><span class="_type-value">Function</span></div>
<div class="_status"><span class="_status-label">Status:</span><span class="_status-value -stable">stable</span></div>
<div class="_platform"><span class="_platform-label">Platform:</span><span class="_platform-value -js">js</span><span class="_platform-value -node">node</span></div>
</div>
<div class="_namespace">@lotsof.sugar.shared.array</div>
<div class="_description">
This function allows you to generate an array from a certain passed quantifier like &quot;2&quot;, &quot;&amp;lt;10&quot;, &quot;&amp;gt;10&quot;, &quot;9-12&quot;, etc...
This is useful to create arrays that are depending on user-friendly quantifier and it&#39;s used for example in the
s-postcss-sugar-plugin for the `@s.lod(&amp;gt;2)` mixin.
</div>

</li>
<li class="_item">

</div>
<div class="_return">
## Return
<ol class="_list">
<li class="_item">
<span class="_description">The generated array</span><span class="_default"></span><span class="_type">Array</span>
</li>
</ol>
</div>
<div class="_examples">
## Example
```js
import { __fromQuantifier } from '@lotsof/sugar/array'
__fromQuantifier(3); // => [0,1,2]
__fromQuantifier('3-5'); // => [3,4,5]
__fromQuantifier('3-6', {
   value(i) => `c-${i}`
}); // => ['c-3','c-4','c-5','c-6']
```
