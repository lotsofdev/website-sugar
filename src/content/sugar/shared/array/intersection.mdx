# intersection
<div class="_metas">
<div class="_type"><span class="_type-label">Type:</span><span class="_type-value">Function</span></div>
<div class="_status"><span class="_status-label">Status:</span><span class="_status-value -stable">stable</span></div>
<div class="_since"><span class="_since-label">Since:</span><span class="_since-value">2.0.0</span></div>
<div class="_platform"><span class="_platform-label">Platform:</span><span class="_platform-value -js">js</span><span class="_platform-value -node">node</span></div>
</div>
<div class="_namespace">@lotsof.sugar.shared.array</div>
<div class="_description">
This function take two arrays and return all the items that are the same.
You can specify if you want only comparing by reference for object, etc... or if
you want to transform items in hashes and compare that instead.
</div>

</li>
<li class="_item">
<span class="_name">settings</span><span class="_default">&lsqb;object Object&rsqb;</span> <span class="_type">Object</span>
<p class="_description">Some settings to refine your comparaison process</p>
</li>
</ol>
</div>
<div class="_return">
## Return
<ol class="_list">
<li class="_item">
<span class="_description">An array containing all the items that are present in all the passed arrays</span><span class="_default"></span><span class="_type">Array</span>
</li>
</ol>
</div>
<div class="_examples">
## Example
