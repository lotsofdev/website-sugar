<li class="_item">

<p class="_description">The instance to check the methods on</p>
</li>
<li class="_item">

</li>
</ol>
</div>
<div class="_return">
## Return
<ol class="_list">
<li class="_item">
<span class="_description">Return true if all is ok, and an array of missing methods if not</span><span class="_default"></span><span class="_type">Boolean&vert;Array</span>
</li>
</ol>
</div>
<div class="_examples">
## Example
```js
class Coco {
   hello() {}
}
import { __methodExists } from '@lotsof/sugar/class';
const myInstance = new Coco();
__methodExists(myInstance, 'hello', 'world'); // => ['world'];
__methodExists(myInstance, 'hello'); // => true
```
