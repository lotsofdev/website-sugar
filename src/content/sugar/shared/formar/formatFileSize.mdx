# formatFileSize
<div class="_metas">
<div class="_type"><span class="_type-label">Type:</span><span class="_type-value">Function</span></div>
<div class="_status"><span class="_status-label">Status:</span><span class="_status-value -stable">stable</span></div>

<div class="_description">
Transform into human readable string a file size from a number (float or integer) or string.
This function use the wonderfull &quot;filesize&quot; npm package under the houd.

</li>
<li class="_item">
<span class="_name">settings</span><span class="_default">&lsqb;object Object&rsqb;</span> <span class="_type">Object</span>
<p class="_description">The &quot;filesize&quot; settings to customize the output&period; See &lsqb;filesize&rsqb;&lpar;https&colon;&sol;&sol;www&period;npmjs&period;com&sol;package&sol;filesize&rpar; settings</p>
</li>
</ol>
</div>
<div class="_return">
## Return
<ol class="_list">
<li class="_item">
<span class="_description">The human readable version of the passed size</span><span class="_default"></span><span class="_type">String</span>
</li>
</ol>
</div>
