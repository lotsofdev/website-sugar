# throttle
<div class="_metas">
<div class="_type"><span class="_type-label">Type:</span><span class="_type-value">Function</span></div>
<div class="_status"><span class="_status-label">Status:</span><span class="_status-value -stable">stable</span></div>
<div class="_since"><span class="_since-label">Since:</span><span class="_since-value">2.0.0</span></div>
<div class="_platform"><span class="_platform-label">Platform:</span><span class="_platform-value -js">js</span><span class="_platform-value -node">node</span></div>
</div>
<div class="_namespace">@lotsof.sugar.shared.function</div>
<div class="_description">
This utils function allows you to make sure that a function that will normally be called
several times, for example during a scroll event, to be called once each threshhold time
</div>
<div class="_params">
## Params
<ol class="_list">
<li class="_item">
<span class="_name">threshhold<span class="_required">*</span></span><span class="_default"></span> <span class="_type">Number</span>
<p class="_description">The delay in ms to wait between two function calls</p>
</li>
<li class="_item">
<span class="_name">fn<span class="_required">*</span></span><span class="_default"></span> <span class="_type">Function</span>
<p class="_description">The function to throttle</p>
</li>
</ol>
</div>
<div class="_return">
## Return
<ol class="_list">
<li class="_item">
<span class="_description">The throttled function</span><span class="_default"></span><span class="_type">Function</span>
</li>
</ol>
</div>
<div class="_examples">
## Example
```js
import { __throttle } from '@lotsof/sugar/function';
const myThrottledFn = __throttle(1000, () => {
		// my function content that will be
		// executed only once each second
});

document.addEventListener('scroll', (e) => {
		// call my throttled function
		myThrottledFn();
});
```
