# stripTags
<div class="_metas">
<div class="_type"><span class="_type-label">Type:</span><span class="_type-value">Function</span></div>
<div class="_status"><span class="_status-label">Status:</span><span class="_status-value -stable">stable</span></div>
<div class="_since"><span class="_since-label">Since:</span><span class="_since-value">2.0.0</span></div>
<div class="_platform"><span class="_platform-label">Platform:</span><span class="_platform-value -js">js</span><span class="_platform-value -node">node</span></div>
</div>
<div class="_namespace">@lotsof.sugar.shared.html</div>
<div class="_description">
Strip tags of an html string.
This is a simple wrapper of the nice &quot;striptags&quot; package that you can find here: https://www.npmjs.com/package/striptags
</div>
<div class="_params">
## Params
<ol class="_list">
<li class="_item">
<span class="_name">html<span class="_required">*</span></span><span class="_default"></span> <span class="_type">String</span>
<p class="_description">The html string to process</p>
</li>
<li class="_item">
<span class="_name">allowedTags<span class="_required">*</span></span><span class="_default"></span> <span class="_type">String&lsqb;&rsqb;</span>
<p class="_description">The tags that are allowed like &lsqb;&apos;a&apos;&comma;&apos;p&apos;&comma;&apos;h1&apos;&comma;&apos;h2&apos;&rsqb;&period;&period;&period;</p>
</li>
<li class="_item">
<span class="_name">tagReplacement<span class="_required">*</span></span><span class="_default"></span> <span class="_type">String</span>
<p class="_description">A string with which you want to replace the tags</p>
</li>
</ol>
</div>
<div class="_return">
## Return
<ol class="_list">
<li class="_item">
<span class="_description">The processed string without tags</span><span class="_default"></span><span class="_type">String</span>
</li>
</ol>
</div>
<div class="_examples">
## Example
```js
import { __stripTags } from '@lotsof/sugar/html'
__stripTags('<p><span>Hello</span> world</p>', '<span>') // <span>Hello</span> world
```
