# isValidNumber
<div class="_metas">
<div class="_type"><span class="_type-label">Type:</span><span class="_type-value">Function</span></div>
<div class="_status"><span class="_status-label">Status:</span><span class="_status-value -stable">stable</span></div>

Check if the passed value is a valid number.
A valid number is either an integer or a number (float).
</div>

## Params
<ol class="_list">
<li class="_item">
<span class="_name">value<span class="_required">*</span></span><span class="_default"></span> <span class="_type">any</span>
<p class="_description">The value to check</p>
</li>
</ol>
</div>
<div class="_return">
## Return
<ol class="_list">
<li class="_item">
<span class="_description">true if it's in a valid number, false if not</span><span class="_default"></span><span class="_type">Boolean</span>
</li>
</ol>
</div>
<div class="_examples">
## Example
```js
import { __isValidNumber } from '@lotsof/sugar/is'
__isValidNumber(12) // true
__isValidNumber('he') // false
__isValidNumber(undefined) // false
__isValidNumber(NaN) // false
```
