# deepClean

<div class="_status"><span class="_status-label">Status:</span><span class="_status-value -stable">stable</span></div>

<div class="_namespace">@lotsof.sugar.shared.object</div>
<div class="_description">
This function takes an object as input and returns a cleaned version of it.
You can pass a cleaner function that has to returns true or false. If returns true, the element will be
keeped. If false, it will be removed from the output.
The default cleaner function remove all that is either null, &quot;&quot; or undefined.
