# deepize
<div class="_metas">
<div class="_type"><span class="_type-label">Type:</span><span class="_type-value">Function</span></div>
<div class="_status"><span class="_status-label">Status:</span><span class="_status-value -stable">stable</span></div>

</div>
<div class="_namespace">@lotsof.sugar.shared.object</div>
<div class="_description">
This function simply take an object like this one:

```js
{
  &#39;something.cool&#39;: &#39;hello&#39;
}
```

and convert it to something like this:

```js
{
  something: {
    cool: &#39;hello&#39;
  }
}
```
</div>

<div class="_examples">
## Example
```js
import { __deepize } from '@lotsof/sugar/object';
__deepize({ 'something.cool': 'hello' }); // => { something: { cool: 'hello' } }
```
