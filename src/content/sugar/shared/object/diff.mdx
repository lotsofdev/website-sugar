# diff
<div class="_metas">
<div class="_type"><span class="_type-label">Type:</span><span class="_type-value">Function</span></div>
<div class="_status"><span class="_status-label">Status:</span><span class="_status-value -stable">stable</span></div>
<div class="_since"><span class="_since-label">Since:</span><span class="_since-value">2.0.0</span></div>
<div class="_platform"><span class="_platform-label">Platform:</span><span class="_platform-value -js">js</span><span class="_platform-value -node">node</span></div>
</div>
<div class="_namespace">@lotsof.sugar.shared.object</div>
<div class="_description">
This function take two objects and return an object that contains only what has been changed between the two.
This function is a simple wrapper around the nice object-diff package from Thomas Jensen that you can find here: https://www.npmjs.com/package/object-diff
</div>
<div class="_params">
## Params
<ol class="_list">
<li class="_item">
<span class="_name">object1<span class="_required">*</span></span><span class="_default"></span> <span class="_type">Object</span>
<p class="_description">The first object used for the diff process</p>
</li>
<li class="_item">
<span class="_name">object2<span class="_required">*</span></span><span class="_default"></span> <span class="_type">Object</span>
<p class="_description">The second object used for the diff process</p>
</li>
<li class="_item">
<span class="_name">settings</span><span class="_default">&lsqb;object Object&rsqb;</span> <span class="_type">Object</span>
<p class="_description">An object of settings to configure the diff process&colon;</p>
</li>
</ol>
</div>
<div class="_return">
## Return
<ol class="_list">
<li class="_item">
<span class="_description">The object that contains only the differences between the two</span><span class="_default"></span><span class="_type">Object</span>
</li>
</ol>
</div>
<div class="_examples">
## Example
```js
import { __diff } from '@lotsof/sugar/object';
const myObject1 = {
   hello: 'world',
   plop: 'yop'
};
const myObject2 = {
   coco: 'plop',
   hello: 'hey!',
   plop: 'yop'
};
__diff(myObject1, myObject2);
{
   coco: 'plop',
   hello: 'hey!'
}
```
