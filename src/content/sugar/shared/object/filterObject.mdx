# filterObject
<div class="_metas">
<div class="_type"><span class="_type-label">Type:</span><span class="_type-value">Function</span></div>
<div class="_status"><span class="_status-label">Status:</span><span class="_status-value -stable">stable</span></div>
<div class="_since"><span class="_since-label">Since:</span><span class="_since-value">2.0.0</span></div>
<div class="_platform"><span class="_platform-label">Platform:</span><span class="_platform-value">js</span><span class="_platform-value">node</span></div>
</div>
<div class="_namespace">@lotsof.sugar.shared.object</div>
<div class="_description">
Allow to filter an object using a function. It works the same as the filter method on the Array object type.
The passed filter function will have as parameter each object properties and must return true or false depending if you want the
passed property in the filtered object
</div>
<div class="_params">
## Params
<ol class="_list">
<li class="_item">
<span class="_name">object<span class="_required">*</span></span><span class="_default"></span> <span class="_type">Object</span>
<p class="_description">The object to filter</p>
</li>
<li class="_item">
<span class="_name">filter<span class="_required">*</span></span><span class="_default"></span> <span class="_type">Function</span>
<p class="_description">The filter function that take as parameter the property itself, and the property name</p>
</li>
</ol>
</div>
<div class="_return">
## Return
<ol class="_list">
<li class="_item">
<span class="_description">The filtered object</span><span class="_default"></span><span class="_type">Object</span>
</li>
</ol>
</div>
<div class="_examples">
## Example
```js
import { __filterObject } from '@lotsof/sugar/object';
__filterObject({
   coco: 'hello',
   plop: true
}, (key, item) => typeof item === 'string');
// { coco: 'hello' }
```
</div>
