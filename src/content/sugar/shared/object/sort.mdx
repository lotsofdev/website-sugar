# sort
<div class="_metas">
<div class="_type"><span class="_type-label">Type:</span><span class="_type-value">Function</span></div>
<div class="_status"><span class="_status-label">Status:</span><span class="_status-value -stable">stable</span></div>
<div class="_since"><span class="_since-label">Since:</span><span class="_since-value">2.0.0</span></div>
<div class="_platform"><span class="_platform-label">Platform:</span><span class="_platform-value -js">js</span><span class="_platform-value -node">node</span></div>
</div>
<div class="_namespace">@lotsof.sugar.shared.object</div>
<div class="_description">
Sort an object properties the same way as the Array.sort do it.
The &quot;a&quot; and &quot;b&quot; argument passed to your sort function will have these properties:
- key: The key of the object
- value: The actual value of the object property
</div>

<div class="_examples">
## Example
```js
import { __sort } from '@lotsof/sugar/object';
__sort({
   coco: { weight: 10 },
   lolo: { weight: 2 },
   plop: { weight: 5 }
}, (a, b) => {
  return a.value.weight - b.value.weight;
});
// {
//   lolo: { weight: 2 },
//   plop: { weight: 5 },
//   coco: { weight: 10 }
// }
```
