# toQueryString
<div class="_metas">
<div class="_type"><span class="_type-label">Type:</span><span class="_type-value">Function</span></div>
<div class="_status"><span class="_status-label">Status:</span><span class="_status-value -stable">stable</span></div>

<div class="_description">
Transform an object (key =&gt; pairs) to a query string like &quot;?var1=value1&amp;var2&quot;
</div>
<div class="_params">
## Params
<ol class="_list">
<li class="_item">
<span class="_name">obj<span class="_required">*</span></span><span class="_default"></span> <span class="_type">Object</span>
<p class="_description">The object to serialize</p>
</li>
</ol>
</div>
<div class="_return">
## Return
<ol class="_list">
<li class="_item">
<span class="_description">The query string</span><span class="_default"></span><span class="_type">String</span>
</li>
</ol>
</div>
<div class="_examples">
## Example
```js
import { __toQueryString } from '@lotsof/sugar/object'
console.log(__toQueryString({
	value1 : 'coco',
	value1 : 'plop'
}));
// => ?value1=coco&value2=plop
```
