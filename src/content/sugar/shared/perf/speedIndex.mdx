# speedIndex
<div class="_metas">
<div class="_type"><span class="_type-label">Type:</span><span class="_type-value">Function</span></div>
<div class="_status"><span class="_status-label">Status:</span><span class="_status-value -stable">stable</span></div>
<div class="_since"><span class="_since-label">Since:</span><span class="_since-value">2.0.0</span></div>
<div class="_platform"><span class="_platform-label">Platform:</span><span class="_platform-value -js">js</span><span class="_platform-value -node">node</span></div>
</div>
<div class="_namespace">@lotsof.sugar.shared.perf</div>
<div class="_description">
This function gives you back an index usually between 0 and 100 that represent
the speed estimation of the computer on which this code runs.
This index is calculated by estimating the loops (for) count that the computer
can make in 100ms and make use of the &quot;slow&quot; and &quot;fast&quot; parameters that represent
an average of how many loops a &quot;slow&quot; computer can do in this timeframe, vs how many loops a &quot;fast&quot;
computer can make in this timeframe.
You can totally update these parameters as you prefer.
Note that these parameters can/will be updated during time to reflect the most
average cases possible.
</div>
<div class="_params">
## Params
<ol class="_list">
<li class="_item">
<span class="_name">slow</span><span class="_default">100000</span> <span class="_type">Number</span>
<p class="_description">How many loops a &quot;slow&quot; computer can make in 100ms</p>
</li>
<li class="_item">
<span class="_name">fast</span><span class="_default">1400000</span> <span class="_type">Number</span>
<p class="_description">How many loops a &quot;fast&quot; computer can make in 100ms</p>
</li>
</ol>
</div>
<div class="_return">
## Return
<ol class="_list">
<li class="_item">
<span class="_description">The speed index calculated from the passed params and the loops the computer can make in 100ms</span><span class="_default"></span><span class="_type">Number</span>
</li>
</ol>
</div>
<div class="_examples">
## Example
```js
import { __speedIndex } from '@lotsof/sugar/perf';
__speedIndex(); // 78
```
