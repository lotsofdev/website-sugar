# countLineChars
<div class="_metas">
<div class="_type"><span class="_type-label">Type:</span><span class="_type-value">Function</span></div>
<div class="_status"><span class="_status-label">Status:</span><span class="_status-value -stable">stable</span></div>
<div class="_since"><span class="_since-label">Since:</span><span class="_since-value">2.0.0</span></div>

<div class="_namespace">@lotsof.sugar.shared.string</div>
<div class="_description">
Count how many characters their is in the passed line.
This function will exclude the characters like the html tags like &lt;red&gt;, etc...
</div>
<div class="_params">
## Params
<ol class="_list">
<li class="_item">
<span class="_name">line<span class="_required">*</span></span><span class="_default"></span> <span class="_type">String</span>
<p class="_description">The line to count</p>
</li>
<li class="_item">
<span class="_name">count</span><span class="_default">&lsqb;object Object&rsqb;</span> <span class="_type">Object</span>
<p class="_description">Specify what you want to count outside of the normal characters of yourse&period; Here&apos;s the list of available options&colon;</p>
</li>
</ol>
</div>
<div class="_return">
## Return
<ol class="_list">
<li class="_item">
<span class="_description">How many characters their is in the line</span><span class="_default"></span><span class="_type">Number</span>
</li>
</ol>
</div>
<div class="_examples">
## Example
```js
import { __countLineChars } from '@lotsof/sugar/string';
 __countLineChars('Hello <red>World</red>'); // 11
```
</div>

<ol class="_list">

</li>
<li class="_item">
<span class="_name">terminalSpecialChars</span><span class="_default">false</span> <span class="_type">Boolean</span>

</li>
</ol>
