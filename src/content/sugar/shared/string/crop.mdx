# crop
<div class="_metas">
<div class="_type"><span class="_type-label">Type:</span><span class="_type-value">Function</span></div>
<div class="_status"><span class="_status-label">Status:</span><span class="_status-value -stable">stable</span></div>
<div class="_since"><span class="_since-label">Since:</span><span class="_since-value">2.0.0</span></div>

<div class="_namespace">@lotsof.sugar.shared.string</div>
<div class="_description">
Allows you to crop a string at a certain length (this length take care of the croping characters like &quot;...&quot;)
</div>

## Params
<ol class="_list">
<li class="_item">
<span class="_name">text<span class="_required">*</span></span><span class="_default"></span> <span class="_type">String</span>
<p class="_description">The text to crop</p>
</li>
<li class="_item">
<span class="_name">length<span class="_required">*</span></span><span class="_default"></span> <span class="_type">Number</span>
<p class="_description">The text length to have after the croping process</p>
</li>
<li class="_item">
<span class="_name">settings</span><span class="_default">&lsqb;object Object&rsqb;</span> <span class="_type">Object</span>
<p class="_description">An object of settings described bellow&colon;</p>
</li>
</ol>
</div>
<div class="_return">
## Return
<ol class="_list">
<li class="_item">
<span class="_description">The cropped text</span><span class="_default"></span><span class="_type">String</span>
</li>
</ol>
</div>
<div class="_examples">
## Example
```js
import { __crop } from '@lotsof/sugar/string';
__crop('Hello World', 10); // => Hello w...
```
</div>
<div class="_settings">
## Settings
<ol class="_list">
<li class="_item">
<span class="_name">chars</span><span class="_default">&period;&period;&period;</span> <span class="_type">String</span>
