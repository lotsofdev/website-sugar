# replaceChunks
<div class="_metas">
<div class="_type"><span class="_type-label">Type:</span><span class="_type-value">Function</span></div>
<div class="_status"><span class="_status-label">Status:</span><span class="_status-value -stable">stable</span></div>
<div class="_since"><span class="_since-label">Since:</span><span class="_since-value">2.0.0</span></div>
<div class="_platform"><span class="_platform-label">Platform:</span><span class="_platform-value">js</span><span class="_platform-value">node</span></div>
</div>
<div class="_namespace">@lotsof.sugar.shared.string</div>
<div class="_description">
Replace the passed &quot;chunks&quot; with the result of the passed replacer function.
The replacer function take as argument the token value
</div>
<div class="_params">
## Params
<ol class="_list">
<li class="_item">
<span class="_name">string<span class="_required">*</span></span><span class="_default"></span> <span class="_type">String</span>
<p class="_description">The string to process</p>
</li>
<li class="_item">
<span class="_name">chunks<span class="_required">*</span></span><span class="_default"></span> <span class="_type">String&lsqb;&rsqb;</span>
<p class="_description">The chunks to replace</p>
</li>
<li class="_item">
<span class="_name">replacer<span class="_required">*</span></span><span class="_default"></span> <span class="_type">Function</span>
<p class="_description">The replacer function that need to return a string with which you want to replace the token</p>
</li>
</ol>
</div>
<div class="_return">
## Return
<ol class="_list">
<li class="_item">
<span class="_description">The new string with replaced chunks</span><span class="_default"></span><span class="_type">String</span>
</li>
</ol>
</div>
<div class="_examples">
## Example
```js
import { __replaceChunks } from '@coffeekraken/sugar/string'
__replaceChunks('hello world', ['or'], (token) => {
     return `<span>${token}</span>;
}) // Hello w<span>or</span>ld
```
